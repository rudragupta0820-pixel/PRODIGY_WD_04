# 💼 Personal Portfolio Website

## 📌 Overview

The Personal Portfolio Website is a modern and responsive web application developed using HTML, CSS, and JavaScript. It serves as a digital portfolio to showcase my skills, education, projects, and contact information in a clean and professional manner.

Designed with a modern dark-themed interface, smooth animations, and interactive elements, the website highlights my journey as a Software Development student while demonstrating practical front-end development skills and responsive design principles.

---

## 🚀 Features

* 🏠 Modern and responsive Home section
* 👨‍💻 Professional About Me section
* 🛠️ Technical Skills showcase
* 📂 Dedicated Projects section with project cards
* 📧 Contact section with contact details and message form
* ✨ Smooth scrolling and typing animation
* 🎨 Modern dark-themed UI with hover effects
* 📱 Fully responsive across desktop, tablet, and mobile devices

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 💡 How It Works

1. Navigate through the website using the fixed navigation bar.
2. Explore the About section to learn about my education and background.
3. View the Skills section showcasing my technical expertise.
4. Browse the Projects section to explore my development work.
5. Visit the Contact section to find my contact information or send a message.
